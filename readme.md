# Setup and run instructions

1. Download this repo from github.
2. Open the folder in your editor of choice, I prefer vscode
    * Optional: Install the Python extension for vscode
    * Optional: Create a virtual environment for this program by typing >Python create environment into the top bar and selecting venv
3. In the terminal enter "pip install -r requirements.txt"
4. Rename one of the settings examples to settings.json
    * Optional: Edit the settings file with additional settings like max_entries
5. Run main.py either in your editor or from the terminal
    * Note: Settings set inline are superceded by those set in settings.json, be aware of the impact this may have on built buylists.
6. Take the freshly generated buylist.csv and upload it at https://www.cardsphere.com/wants



# Safety and Warnings
Please try to pass human eyes over your buylist and do a thorough sanity check before taking it live. This software is a draft. I've used it and similar myself, but you aren't me and I don't know what impact losing $20 overpaying on foil Basking Rootwallas may have on you. Cardsphere will pause your wants on upload, but a sanity check is always a good idea. The license this software is presented with doesn't hold me responsible for poor decisions you make with it, but I'll still feel very bad if you hurt yourself with it.
