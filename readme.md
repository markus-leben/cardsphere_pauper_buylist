# Setup and run instructions

1. Download this repo from github.
2. Open the folder in your editor of choice, I prefer vscode
    * Optional: Install the Python extension for vscode
    * Optional: Create a virtual environment for this program by typing >Python create environment into the top bar and selecting venv
3. In the terminal enter "pip install -r requirements.txt"
4. Rename one of the settings examples to settings.json
    * Optional: Edit the settings file with additional settings like max_entries
5. Run main.py either in your editor or from the terminal
